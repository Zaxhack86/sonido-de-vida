# ✝ Sonido de Vida

> Biblia gratuita en audio. Lee, escucha y descarga cualquier versículo en español.

🌐 [sonidodevida.com](https://sonidodevida.com)

## ✨ Características

- 🔍 Buscador de versículos por referencia (Libro, Capítulo, Versículo)
- 🎲 Versículo aleatorio con un clic
- 🔊 Reproducción de audio con Web Speech API (voz en español)
- 💾 Descarga de versículos como archivo HTML autocontenido
- ✅ 100% gratuito, sin registro, sin límites
- 📱 Diseño responsive para móvil y desktop

## 🛠️ Tecnologías

- HTML5, CSS3, JavaScript (Vanilla - sin frameworks)
- Spanish Bible API (Reina Valera 1960)
- Web Speech API para síntesis de voz en español
- Hosting en Vercel con CDN global
